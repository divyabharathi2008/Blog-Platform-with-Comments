Blog Platform with Comments

Project Overview

A simple Blog Platform developed using HTML, CSS, and JavaScript. Users can create blog posts and add comments dynamically through an interactive web interface.

Features

- Create Blog Posts
- Publish Blog Content
- Add Comments to Posts
- Dynamic Content Updates
- Input Validation
- User-Friendly Interface

Technologies Used

- HTML
- CSS
- JavaScript

Project Structure

- index.html
- style.css
- script.js

How It Works

1. Enter a blog title.
2. Write blog content.
3. Click the "Publish" button.
4. The blog post will appear on the page.
5. Add comments using the comment input box.
6. Click the "Comment" button to display comments under the post.
